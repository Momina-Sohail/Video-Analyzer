Cryptographic webhook signatures

Ensure absolute trust between your custom background worker service and Next.js backend endpoint using SHA-256 header validation tokens.

const signature = req.headers['x-aura-signature'];
const hash = crypto.createHmac('sha256', process.env.WEBHOOK_SECRET)
.update(JSON.stringify(payload))
.digest('hex');
if (hash !== signature) {
return NextResponse.json({ error: 'Invalid token' }, { status: 401 });

DEVELOPER DOCUMENTATION

Integrate in minutes.

Interact with Aura Video Analyzer via clean REST API request definitions or custom Javascript SDK libraries.

Create a processing job

POST /api/jobs

Initialize a background pipeline execution. Provide a direct MP4 storage reference url or a public platform URL (e.g. YouTube).

Request Headers:
Authorization: Bearer AURA_SECRET_KEY
Content-Type: application/json
Request Payload:
{
  "source_url": "https://youtube.com/watch?v=...",
  "webhook_url": "https://myapp.com/webhook"
}
cURL Node.js 
curl -X POST https://api.aura.video/v1/jobs \
  -H "Authorization: Bearer $AURA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "source_url": "https://youtube.com/watch?v=dQw",
    "webhook_url": "https://my.app/api/webhook"
  }'
            
const fetch = require('node-fetch');

const response = await fetch('https://api.aura.video/v1/jobs', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + process.env.AURA_KEY,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    source_url: 'https://youtube.com/watch?v=dQw',
    webhook_url: 'https://my.app/api/webhook'
  })
});

const data = await response.json();
console.log(data.job_id);
